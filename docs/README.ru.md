# AI Work Intelligence Platform — Документация

> Каноничная версия — английская (`README.md`); это поддерживаемая русская копия.
> Соглашение по языкам (D7): **EN каноничен** (`*.md`), RU — поддерживаемая копия (`*.ru.md`).

Набор документации, превращающий рабочие обсуждения из Telegram в структурированные Work Items
через human-in-the-loop review.

английский) плюс `docs/<name>.ru.md` (поддерживаемая русская копия). Подкаталогов `en/` или `ru/` нет.
